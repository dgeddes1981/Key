setup examples
    This is a great way to see the sorts of things Key is capable of (from an Administrator point of view).

    There isn't really a 'default' setup for Key when it's first booted. These setup files configure Key to a sample talker, including rooms, objects and resident commands.

    If you're interested to see the power of Key, take a look at the commands used to set it up.
