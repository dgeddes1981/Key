Beta release 0.91, 21st October, 1998 (Paul):

At the moment, Key doesn't have any of the vision applications available.

That said, most of the server itself is already in-place.  The server has been developed and tested with an extremely advanced virtual community application, called a talker.  A MUD you may have heard of, a MUD is a game, just like Ultima Online, only generally text-based and free.  A talker looks similar to a MUD, but does not have the gaming aspect.  Instead, a talker has much more support for user-interaction and grouping.  I give up, you'll just have to try it.  I used to be involved in a talker called Forest, you may want to have a look at it, as an example of what I mean.

In any case, this talker application has been used to build and test all aspects of the server (each of the applications above only use some part of what Key can do, the talker can be configured to use nearly all of them).  You are welcome to download it and take a look at it, but the rest of this page is aimed at those interested in using Key because it is a talker, and not because it can do everything I listed above.
