cvs
    We now have an anonymous CVS server you can get the latest and greatest key code from.
    • User: anoncvs
    • Password: anoncvs
    • Machine: ftoomsh.progsoc.uts.edu.au
    • CVS root: /home/projects/cvsroot
    • CVS module: key
    Using the command line CVS, this looks something like: 
      export CVSROOT=:pserver:anoncvs@ftoomsh.progsoc.uts.edu.au:/home/projects/cvsroot
cvs login
      Enter 'anoncvs' as the password
      cvs -z3 checkout key

      For windows users, I've written a little blurb to get you started quickly:

      Do this:

      • Download: cvs-1.10-win.zip
      • Use WinZip to extract it to C:\cvs
      • Open a DOS prompt. Type:
        SET PATH=%PATH%;C:\CVS
SET CVSROOT=:pserver:anoncvs@ftoomsh.progsoc.uts.edu.au:/home/projects/cvsroot
cvs login
        Use 'anoncvs' for the password 
        C:
mkdir C:\key
cd C:\
cvs -z3 checkout key
      • A new directory, C:\key has been created containing all of the source files for key.
